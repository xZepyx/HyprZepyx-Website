About HyprZepyx

A comprehensive Hyprland setup that transforms your desktop experience

What is HyprZepyx?

HyprZepyx is a meticulously crafted Hyprland configuration that brings together the best of modern Linux desktop environments. It combines stunning visual effects, smooth animations, and a highly productive workflow into one cohesive package.

Philosophy

Built for power users who refuse to compromise between aesthetics and functionality. Every animation, keybinding, and visual element has been carefully tuned to provide an exceptional user experience.

Key Components

  • Hyprland: Dynamic tiling Wayland compositor
  • Waybar: Highly customizable status bar
  • Rofi: Beautiful application launcher
  • Dunst: Lightweight notification daemon
  • Kitty: GPU-accelerated terminal emulator

Installation Guide

Get up and running with HyprZepyx in minutes

01

Prerequisites

Make sure you have Hyprland installed and backup your current config.

cp -r ~/.config/ ~/.config/conf.bak
02

Clone Repository

Clone HyprZepyx and install dependencies.

git clone https://github.com/xZepyx/HyprZepyx.git ~/hyprzepyx
cd ~/hyprzepyx
bash deps-installer.sh
03

Setup Configuration

Copy configs and set permissions.

cp -r ~/.config/hypr/configs/* ~/.config/
chmod +x ~/.config/hypr/scripts/*
04

Launch Hyprland

Start your new desktop environment.

Hyprland

Requirements

  • Arch Linux (recommended)
  • Hyprland compositor
  • 4GB RAM (Minimum)
  • Intel I3 or AMD R3 Processor
