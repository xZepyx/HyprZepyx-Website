Installation Guide

Get HyprZepyx up and running in minutes

Prerequisites

Ensure you have the following dependencies installed:

sudo pacman -S kitty waybar swww swaync eww rofi hyprland

Or use the automated installer:

bash deps-installer.sh
1

Clone the Repository

Download HyprZepyx to your local machine:

git clone https://github.com/xZepyx/HyprZepyx.git ~/.config/hyprzepyx
cd ~/.config/hyprzepyx
2

Install Dependencies

Run the prerequisites script to install all required packages:

bash prerequisites.sh
3

Copy Configuration Files

Copy the Hyprland configuration to your config directory:

cp -r config/* ~/.config/
4

Launch Hyprland

Start your new HyprZepyx desktop:

hyprland

Pro Tips

  • Adjust paths in scripts if you have a custom setup directory
  • Install recommended GTK themes and fonts for the best visual experience
  • Use Super + D to open the app launcher after installation
  • Check the troubleshooting section if you encounter any issues

Troubleshooting

Common Issues

  • • Waybar icons missing: Install FontAwesome or Material Icons
  • • Eww widgets missing: Run eww daemon before Hyprland
  • • Rofi launcher errors: Check config.rasi paths
  • • Hyprland crashes: Check ~/.local/share/hyprland.log

Quick Fixes

  • • Restart Waybar: killall waybar && waybar &
  • • Reload Hyprland: hyprctl reload
  • • Reset wallpaper: swww img /path/to/wallpaper
  • • Check logs: journalctl -u hyprland

Project Story

HyprZepyx was born from a desire to create the perfect desktop environment that doesn't compromise on speed, functionality, or visual appeal. Starting as a personal configuration, it evolved into a comprehensive Hyprland setup that embodies the philosophy of "Fatal Exception : Elegance."

The project focuses on three core principles: Speed - lightning-fast performance even on older hardware, Clarity - intuitive design that gets out of your way, and Aesthetics - beautiful visuals that inspire productivity.

Each of the four themes represents a different mood and workflow style, from the vibrant gradients of Spectral-Horizon to the minimal elegance of Abyss. The project continues to evolve with community feedback and new Hyprland features.
